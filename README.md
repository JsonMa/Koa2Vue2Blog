# Koa2Vue2Blog
This is my blog project written with Koa2 and Vue2.
## 博客系统技术栈
### 前端
前端框架用的是Vue2.0
前后端通信用的fetch
评论系统用的是多说
界面风格模仿了vue博客
博客编辑器采用的是SimpleMDE
### 后端
后端框架用的是Koa2
数据库用的是Mongodb
数据库操作用的是Mongoose
登录用的是jwt
后端环境变量的设置使用的是[dotenv](https://github.com/motdotla/dotenv "dotenv github地址")
### 说明
    后端所有语法均采用的是ES6语法，采用babel进行转译
